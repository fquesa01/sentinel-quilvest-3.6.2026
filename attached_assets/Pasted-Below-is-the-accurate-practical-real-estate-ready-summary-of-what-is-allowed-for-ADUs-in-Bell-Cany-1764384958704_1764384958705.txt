✅ MAXIMUM ALLOWED ADUs IN BELL CANYON (Ventura County + BCA Rules)

Bell Canyon is unusually flexible because the lots are large, rural-residential, and zoned for equestrian use.

You are allowed:

1. One Detached ADU

Maximum Size (Ventura County → applies to Bell Canyon)

Up to 1,200 sq ft

This is the largest size Ventura County allows by right.

No minimum lot size requirement.

Height Limit

25 feet max (2 stories allowed) if the ADU is more than 5 feet from the main structure

Must comply with BCA guidelines:

Roof pitch must match or be harmonious

Exterior colors consistent

Max two stories if it does not violate view corridors

Setbacks

4 ft side and rear yard setbacks (state ADU law supersedes local CC&Rs)

BCA will still apply architectural review but cannot block a compliant ADU.

Uses Allowed

Long-term residential

Guest house

Office/studio

Gym

Caretaker suite

In-law suite

Teen suite

Not Allowed

Short-term rentals (<30 days) per BCA rules

Commercial use

Separate parcel/condo conversion

2. One Junior ADU (JADU)

Must be within the existing main residence

Max 500 sq ft

Must have a separate entrance

Must have a kitchenette

Can share a bathroom

Most owners do not build these because the detached 1,200 sq ft ADU is more valuable.

🔥 Total Maximum Accessory Units on One Lot

You may legally have:

✔ 1 × Detached ADU (up to 1,200 sq ft)
✔ 1 × Junior ADU (up to 500 sq ft)
Total: 2 additional dwelling spaces

Almost all Bell Canyon large-lot owners build only the detached version.

📏 Height Rules (Precise)

Bell Canyon follows Ventura County height rules for ADUs:

If Detached ADU

16 ft minimum guaranteed height allowed by state law

Ventura County allows up to 25 ft for ADUs if:

It’s on a lot with an existing two-story home, OR

It is built over a garage, OR

It matches the architectural character

BCA will review the aesthetic only, not the height itself unless excessive.

🧱 Setback Rules

4 ft from side and rear yards (state law)

Front yard setbacks follow main structure setbacks (typically 20–25 ft)

BCA may request some screening (trees, shrubs) but cannot deny a compliant ADU.

💧 Utility + Infrastructure Rules

Must connect to existing septic or sewer (Bell Canyon uses sewer — easier)

Separate utility meters are optional

Fire sprinklers not required in ADU if the main house doesn’t have them

🎨 Exterior Requirements (Bell Canyon HOA Rules)

BCA requires:

Materials consistent with the main residence

Neutral earth-tone color palette

Roofing compatible with the main home

Non-reflective glass

Proper landscaping to screen structures from neighbors

Site plan + elevations for approval

They cannot deny an ADU outright if it meets state law.
