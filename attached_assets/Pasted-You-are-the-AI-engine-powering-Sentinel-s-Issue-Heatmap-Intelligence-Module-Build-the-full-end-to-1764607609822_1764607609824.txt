1. DATA INGESTION AND ANALYSIS ENGINE
====================================================

For every ingested communication (email, text, chat, document, attachment), extract:

A. TOPIC CLASSIFICATION
Use hybrid models (LLM classification + embedding clustering).
Default topics (extensible):
- Legal  
- HR  
- Contracts  
- Finance  
- Safety  
- Sensitive  
- General  
- Custom case-level topics

B. ISSUE EXTRACTION
Extract the central issue using entity detection + LLM summarization.
Include:
- Event mentions  
- Deadlines  
- Contract numbers  
- People and organizations  
- References to places, products, safety issues, HR concerns, etc.

C. URGENCY & IMPORTANCE LANGUAGE
Calculate an urgency_score (0–10) using weighted detection of:
High urgency keywords:
urgent, critical, important, immediate, escalate, asap, emergency, crisis, deadline, right away

Moderate urgency:
please advise, following up, need clarification, check in

Escalation sentiments:
“don’t forward,”  
“take offline,”  
“delete after reading,”  
“shouldn’t put this in writing,”  
“we need to talk now,”  
fear signals, blame language, legal worry, etc.

D. SENTIMENT ANALYSIS
Assign numeric sentiment score (-1 negative to +1 positive).

E. ENTITY EXTRACTION
Extract:
- People (internal & external)
- Organizations/domains
- Projects or case items
- Contract names
- Facilities/locations
- Document identifiers

F. ISSUE OBJECT (STORE THIS OBJECT)
