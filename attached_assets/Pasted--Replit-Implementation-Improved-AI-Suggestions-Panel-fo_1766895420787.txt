# Replit Implementation: Improved AI Suggestions Panel for Ambient Intelligence

Improve the AI Suggestions panel in the Ambient Intelligence feature to be glanceable during live conversations. Users are cognitively loaded during meetings and need to make quick decisions about documents without reading lengthy text.

## Task 1: Create the new SuggestionCard component

Create file: `components/ambient/SuggestionCard.tsx`

```tsx
import { useState } from 'react';

interface DocumentResult {
  id: string;
  title: string;
  type: 'email' | 'document';
  sender?: string;
  date: string;
  preview: string;
}

interface SuggestionCardProps {
  suggestion: {
    id: string;
    topic: string;
    triggerQuote: string;
    confidence: 'high' | 'medium' | 'low';
    results: DocumentResult[];
    status: 'found' | 'searching' | 'no_results';
  };
  onViewDocument: (docId: string) => void;
  onDismiss: () => void;
  onDismissDocument: (docId: string) => void;
}

export function SuggestionCard({
  suggestion,
  onViewDocument,
  onDismiss,
  onDismissDocument
}: SuggestionCardProps) {
  const [expandedDoc, setExpandedDoc] = useState<string | null>(null);
  const [summarizing, setSummarizing] = useState<string | null>(null);
  const [summaries, setSummaries] = useState<Record<string, string>>({});

  const handleSummarize = async (docId: string) => {
    setSummarizing(docId);
    try {
      const response = await fetch(`/api/documents/${docId}/summarize`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ context: suggestion.triggerQuote })
      });
      const { summary } = await response.json();
      setSummaries(prev => ({ ...prev, [docId]: summary }));
      setExpandedDoc(docId);
    } catch (error) {
      console.error('Summarization failed:', error);
    }
    setSummarizing(null);
  };

  // Searching state
  if (suggestion.status === 'searching') {
    return (
      <div className="suggestion-card searching">
        <div className="searching-content">
          <div className="pulse-ring" />
          <span className="searching-topic">{suggestion.topic}</span>
        </div>
      </div>
    );
  }

  if (suggestion.results.length === 0) return null;

  return (
    <div className="suggestion-card">
      {/* Header - What triggered this */}
      <div className="card-header">
        <div className="trigger-indicator">
          <span className="quote-icon">"</span>
          <span className="trigger-text">{suggestion.triggerQuote}</span>
        </div>
        <button className="dismiss-card" onClick={onDismiss} aria-label="Dismiss">
          <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
            <line x1="18" y1="6" x2="6" y2="18"/>
            <line x1="6" y1="6" x2="18" y2="18"/>
          </svg>
        </button>
      </div>

      {/* Topic with confidence indicator */}
      <div className="topic-row">
        <span className={`confidence-dot ${suggestion.confidence}`} />
        <span className="topic-label">{suggestion.topic}</span>
        <span className="result-count">{suggestion.results.length}</span>
      </div>

      {/* Document list */}
      <div className="documents-list">
        {suggestion.results.map((doc) => (
          <div key={doc.id} className={`document-item ${expandedDoc === doc.id ? 'expanded' : ''}`}>
            <div className="document-row">
              {/* Type icon */}
              <div className={`doc-type-icon ${doc.type}`}>
                {doc.type === 'email' ? (
                  <svg width="14" height="14" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                    <rect x="2" y="4" width="20" height="16" rx="2"/>
                    <path d="M22 6L12 13L2 6"/>
                  </svg>
                ) : (
                  <svg width="14" height="14" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                    <path d="M14 2H6a2 2 0 0 0-2 2v16a2 2 0 0 0 2 2h12a2 2 0 0 0 2-2V8z"/>
                    <polyline points="14 2 14 8 20 8"/>
                  </svg>
                )}
              </div>

              {/* Document info */}
              <div className="doc-info">
                <span className="doc-title">{doc.title}</span>
                {doc.sender && <span className="doc-meta">{doc.sender}</span>}
              </div>

              {/* Action buttons */}
              <div className="quick-actions">
                <button 
                  className="action-btn view"
                  onClick={() => onViewDocument(doc.id)}
                  title="View"
                >
                  <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                    <path d="M1 12s4-8 11-8 11 8 11 8-4 8-11 8-11-8-11-8z"/>
                    <circle cx="12" cy="12" r="3"/>
                  </svg>
                </button>
                <button 
                  className="action-btn summarize"
                  onClick={() => handleSummarize(doc.id)}
                  disabled={summarizing === doc.id}
                  title="Summarize"
                >
                  {summarizing === doc.id ? (
                    <div className="mini-spinner" />
                  ) : (
                    <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                      <line x1="21" y1="10" x2="3" y2="10"/>
                      <line x1="21" y1="6" x2="3" y2="6"/>
                      <line x1="21" y1="14" x2="3" y2="14"/>
                      <line x1="21" y1="18" x2="10" y2="18"/>
                    </svg>
                  )}
                </button>
                <button 
                  className="action-btn skip"
                  onClick={() => onDismissDocument(doc.id)}
                  title="Skip"
                >
                  <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                    <polyline points="13 17 18 12 13 7"/>
                    <polyline points="6 17 11 12 6 7"/>
                  </svg>
                </button>
              </div>
            </div>

            {/* Expanded summary */}
            {expandedDoc === doc.id && summaries[doc.id] && (
              <div className="document-summary">
                <div className="summary-label">
                  <svg width="12" height="12" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                    <polygon points="12 2 15.09 8.26 22 9.27 17 14.14 18.18 21.02 12 17.77 5.82 21.02 7 14.14 2 9.27 8.91 8.26 12 2"/>
                  </svg>
                  AI Summary
                </div>
                <p className="summary-text">{summaries[doc.id]}</p>
                <button 
                  className="view-full-btn"
                  onClick={() => onViewDocument(doc.id)}
                >
                  View Full Document →
                </button>
              </div>
            )}
          </div>
        ))}
      </div>
    </div>
  );
}
```

## Task 2: Create the updated AISuggestionsPanel component

Create or replace file: `components/ambient/AISuggestionsPanel.tsx`

```tsx
import { SuggestionCard } from './SuggestionCard';

interface DocumentResult {
  id: string;
  title: string;
  type: 'email' | 'document';
  sender?: string;
  date: string;
  preview: string;
}

interface Suggestion {
  id: string;
  topic: string;
  triggerQuote: string;
  confidence: 'high' | 'medium' | 'low';
  results: DocumentResult[];
  status: 'found' | 'searching' | 'no_results' | 'dismissed';
}

interface AISuggestionsPanelProps {
  suggestions: Suggestion[];
  onViewDocument: (docId: string) => void;
  onDismissSuggestion: (suggestionId: string) => void;
  onDismissDocument: (suggestionId: string, docId: string) => void;
}

export function AISuggestionsPanel({
  suggestions,
  onViewDocument,
  onDismissSuggestion,
  onDismissDocument
}: AISuggestionsPanelProps) {
  const activeSuggestions = suggestions.filter(s => 
    s.status !== 'dismissed' && (s.status === 'searching' || s.results.length > 0)
  );

  const totalDocs = activeSuggestions.reduce((sum, s) => sum + (s.results?.length || 0), 0);

  return (
    <div className="ai-suggestions-panel">
      {/* Header */}
      <div className="panel-header">
        <div className="header-left">
          <span className="ai-icon">✨</span>
          <span className="header-title">AI Suggestions</span>
        </div>
        {totalDocs > 0 && (
          <div className="doc-counter">
            <span className="counter-number">{totalDocs}</span>
            <span className="counter-label">found</span>
          </div>
        )}
      </div>

      {/* Suggestions list */}
      <div className="suggestions-stream">
        {activeSuggestions.length === 0 ? (
          <div className="empty-state">
            <div className="empty-icon">
              <svg width="32" height="32" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="1.5">
                <circle cx="12" cy="12" r="10"/>
                <path d="M8 14s1.5 2 4 2 4-2 4-2"/>
                <line x1="9" y1="9" x2="9.01" y2="9"/>
                <line x1="15" y1="9" x2="15.01" y2="9"/>
              </svg>
            </div>
            <p className="empty-text">Listening for relevant topics...</p>
          </div>
        ) : (
          activeSuggestions.map(suggestion => (
            <SuggestionCard
              key={suggestion.id}
              suggestion={suggestion}
              onViewDocument={onViewDocument}
              onDismiss={() => onDismissSuggestion(suggestion.id)}
              onDismissDocument={(docId) => onDismissDocument(suggestion.id, docId)}
            />
          ))
        )}
      </div>

      {/* Footer */}
      {totalDocs > 0 && (
        <div className="panel-footer">
          <button className="footer-action">
            <svg width="14" height="14" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
              <rect x="3" y="3" width="18" height="18" rx="2" ry="2"/>
              <line x1="9" y1="3" x2="9" y2="21"/>
            </svg>
            View All in Split
          </button>
        </div>
      )}
    </div>
  );
}
```

```typescript
import { NextApiRequest, NextApiResponse } from 'next';
import Anthropic from '@anthropic-ai/sdk';

export default async function handler(req: NextApiRequest, res: NextApiResponse) {
  if (req.method !== 'POST') {
    return res.status(405).json({ error: 'Method not allowed' });
  }

  const { id } = req.query;
  const { context } = req.body;

  try {
    // Fetch document content from your database
    // Replace this with your actual document fetching logic
    const document = await fetchDocumentById(id as string);
    
    if (!document) {
      return res.status(404).json({ error: 'Document not found' });
    }

    const anthropic = new Anthropic({
      apiKey: process.env.ANTHROPIC_API_KEY
    });

    const response = await anthropic.messages.create({
      model: 'claude-sonnet-4-20250514',
      max_tokens: 200,
      messages: [{
        role: 'user',
        content: `You are helping a lawyer during a live meeting. They heard "${context}" and need to quickly understand if this document is relevant.

DOCUMENT: ${document.title}
CONTENT (first 3000 chars):
${document.content.substring(0, 3000)}

Provide a 2-3 sentence summary that:
1. States what this document IS (contract, email, invoice, etc.)
2. Highlights anything relevant to "${context}"
3. Notes any red flags or key dates

Be direct. No preamble. Start with "This is a..."`
      }]
    });

    const summary = response.content[0].type === 'text' 
      ? response.content[0].text 
      : '';

    return res.json({ summary });
  } catch (error) {
    console.error('Summarization error:', error);
    return res.status(500).json({ error: 'Failed to summarize' });
  }
}
```

## Summary

This implementation provides:

1. **Glanceable design** - Color dots for confidence, icons for doc types, truncated text
2. **Three quick actions per document**:
   - 👁 View (blue) - Opens document
   - 📝 Summarize (green) - AI summary expands inline
   - ⏭ Skip (gray) - Dismisses that document
3. **Trigger context** - Shows what was said that triggered the suggestion
4. **Inline AI summaries** - 2-3 sentence contextual summaries without leaving the panel
5. **Clean visual hierarchy** - Most important info is immediately visible

The user can now glance at the panel during conversation and make quick decisions without reading walls of text.
