3. Backend API Logic (Express + Prisma Example)
GET /api/reports/:caseId/discovery
This endpoint:
1.	Validates RBAC (admin, firm_user, or client for their own case)
2.	Runs the full set of analytics queries
3.	Bundles into one JSON or downloadable file (PDF/Excel/CSV)
Example Code Skeleton
router.get(
  "/reports/:caseId/discovery",
  authMiddleware,
  requireCaseAccess,
  async (req, res) => {
    const caseId = req.params.caseId;

    const documents = await prisma.document.findMany({
      where: { caseId },
      include: { tags: true, privilegeTags: true }
    });

    const communications = await prisma.communication.findMany({
      where: { caseId }
    });

    const batches = await prisma.productionBatch.findMany({
      where: { caseId }
    });

    const custodians = groupBy(documents, "custodian");
    const fileTypes = groupBy(documents, "fileType");
    const tagCounts = countTags(documents);
    const privilegeStats = countPrivilege(documents);
    const commMap = buildCommunicationGraph(communications);

    const report = {
      metadata: computeMetadata(documents),
      custodians: computeCustodianSummary(custodians),
      communications: commMap,
      tagAnalysis: tagCounts,
      fileTypes,
      privilege: privilegeStats,
      production: batches,
      inventory: documents
    };

    res.json(report);
  }
);

You can render this with Handlebars and print via Puppeteer:
// server/reports/generateDiscoveryPdf.ts
import puppeteer from "puppeteer";
import handlebars from "handlebars";
import fs from "fs/promises";
import path from "path";

export async function generateDiscoveryPdf(reportData: any): Promise<Buffer> {
  const templatePath = path.join(__dirname, "templates", "discoveryReport.html");
  const htmlSource = await fs.readFile(templatePath, "utf8");
  const template = handlebars.compile(htmlSource);
  const html = template(reportData);

  const browser = await puppeteer.launch({ headless: true });
  const page = await browser.newPage();
  await page.setContent(html, { waitUntil: "networkidle0" });
  const pdf = await page.pdf({ format: "Letter", printBackground: true });
  await browser.close();
  return pdf;
}

export async function buildDiscoveryWorkbook(report: any): Promise<ExcelJS.Workbook> {
  const wb = new ExcelJS.Workbook();
  wb.creator = "Sentinel Counsel LLP";
  wb.created = new Date();

  const headerStyle = {
    font: { bold: true, color: { argb: "FFFFFFFF" } },
    fill: { type: "pattern", pattern: "solid", fgColor: { argb: "FF111827" } },
    alignment: { vertical: "middle", horizontal: "left" }
  };

  const thinBorder = {
    top: { style: "thin", color: { argb: "FF1F2937" } },
    left: { style: "thin", color: { argb: "FF1F2937" } },
    bottom: { style: "thin", color: { argb: "FF1F2937" } },
    right: { style: "thin", color: { argb: "FF1F2937" } }
  };

  /* SUMMARY SHEET */
  const summary = wb.addWorksheet("Summary", { views: [{ state: "frozen", ySplit: 2 }] });
  summary.columns = [
    { header: "Metric", key: "metric", width: 32 },
    { header: "Value", key: "value", width: 25 },
    { header: "Notes", key: "notes", width: 40 }
  ];
  summary.getRow(1).eachCell(cell => {
    cell.style = { ...cell.style, ...headerStyle, border: thinBorder };
  });

________________________________________
1. What counts as a “document” in Sentinel?
For your Logicull-style report, we’ll treat these as the core “documents”:
•	communications
o	Emails, chats, etc. This already has:
	subject, body, sender, recipients
	communicationType, sourceType
	custodianId, custodianName, custodianDepartment
	caseId
	documentHash, nearDuplicateHash
	fileSize, filePath, fileExtension, mimeType
	privilegeStatus, isRedacted, batesNumber, batesRange
	productionSetId, productionStatus (not_produced, produced, withheld, etc.)
•	custodians – custodian profiles for each case.
•	documentCodings – responsiveness & privilege coding:
o	documentId (→ communications.id)
o	responsiveness (responsive, non_responsive, etc.)
o	isPrivileged, privilegeType, isHot, confidentialityLevel
•	tags + documentTags – generic tagging:
o	documentTags.entityType = "communication"
o	documentTags.entityId = communications.id
•	productionSets – production batches/sets.
•	privilegeLogs – formal privilege log entries per case.
So the Discovery Report will be driven almost entirely by:
cases, communications, custodians, documentCodings, tags, documentTags, productionSets, privilegeLogs.
________________________________________
2. Mapping report sections → your tables/columns
A. Metadata
•	Case info
o	caseId → cases.id
o	caseNumber → cases.caseNumber
o	caseName → cases.title
o	violationType → cases.violationType
•	Document totals (for that case):
o	totalDocuments → count of communications where communications.caseId = cases.id
o	processedDocuments → same for now (you can later add a processed flag)
•	Responsive:
o	from documentCodings where documentCodings.documentId IN docIds && responsiveness = 'responsive'
•	Privileged:
o	from documentCodings.isPrivileged = 'true' or
o	communications.privilegeStatus != 'none'
•	Produced / withheld:
o	from communications.productionStatus:
	produced → productionStatus = 'produced'
	withheld → productionStatus = 'withheld'
•	Date range:
o	earliestDate / latestDate → MIN/MAX of communications.timestamp for that case.
________________________________________
B. Custodian Summary
Use both communications and custodians:
•	Group communications by custodianId (or custodianName if null).
•	Join custodians on custodians.id = communications.custodianId to get:
o	fullName, department, location, etc.
•	For each custodian:
o	totalDocs = count communications
o	responsive = docs whose documentCodings.responsiveness = 'responsive'
o	privileged = docs with isPrivileged = true or privilegeStatus != 'none'
o	produced = docs with productionStatus = 'produced'
________________________________________
C. Tags & Issue Coding
•	tags table:
o	name, category, color
•	documentTags:
o	entityType = 'communication'
o	entityId = communications.id
•	Join documentTags → tags to compute:
o	For each tag: count, % of docs, who applied it (documentTags.taggedBy).
________________________________________
D. Privilege Summary
•	From documentCodings:
o	isPrivileged, privilegeType (enum: attorney_client_privileged, work_product, both)
•	From communications.privilegeStatus as another signal
•	From privilegeLogs:
o	For the case: privilegeLogs.caseId = cases.id.
You can classify:
•	AC: docs with privilegeType = 'attorney_client_privileged' or privilegeStatus = 'attorney_client_privileged'
•	WP: work_product
•	Both: both
•	Risk: anything in privilegeLogs with isPartiallyPrivileged = true or redactionApplied = true or docs where privileged but productionStatus = 'produced' → possible inadvertent.
________________________________________
E. File Types
•	From communications:
o	fileExtension, mimeType, fileSize
•	Optionally combine with ingestionFiles if you want ingest-level file types (fileType, fileSize, metadata).
Aggregate:
•	type = fileExtension (fallback to communicationType or mimeType if missing)
•	count, total size, a few sample file names.
________________________________________
F. Production Batches
•	From productionSets (this is your “ProductionBatch”):
o	id, productionNumber, productionName
o	exportFormat, renditionType, batesPrefix, batesStartNumber/EndNumber
o	documentIds (jsonb array) and documentCount
o	hashManifest (document-level hashes)
o	transmittedAt, transmittedTo, exportPackageUrl
•	Link each production set back to communications via:
o	communications.productionSetId = productionSets.id
o	and productionStatus per doc (produced vs withheld).
Then for each production set:
•	Docs Produced = # comms where productionSetId = ps.id && productionStatus = 'produced'
•	Withheld = productionStatus = 'withheld'
•	Redacted = isRedacted = 'true' or privilegeLogs.redactionApplied.
________________________________________
G. Inventory (Logicull-style row)
Use communications as the base:
Columns like:
•	Doc ID → communications.id
•	Case ID → communications.caseId
•	Custodian → communications.custodianName (or join custodians.fullName)
•	Sender → communications.sender
•	Recipients → communications.recipients (JSON array)
•	Subject → communications.subject
•	Date → communications.timestamp
•	File Name → communications.filePath last path segment or from metadata
•	File Path → communications.filePath
•	File Extension → communications.fileExtension
•	Mime Type → communications.mimeType
•	Size → communications.fileSize
•	Hash → communications.documentHash
•	Near Dup Hash → communications.nearDuplicateHash
•	Thread / Family → communications.emailThreadId, communications.documentFamilyId
•	Production Status → communications.productionStatus
•	Bates → communications.batesNumber, communications.batesRange
•	Tags → from documentTags + tags
•	Privilege Status → communications.privilegeStatus

3. Concrete buildDiscoveryReport using Drizzle
Here’s a focused implementation that matches your schema.
// src/server/reports/buildDiscoveryReport.ts
import { db } from "@/server/db"; // adjust path to your drizzle instance
import {
  cases,
  communications,
  custodians,
  tags,
  documentTags,
  documentCodings,
  productionSets,
  privilegeLogs,
} from "@/server/db/schema";
import { and, eq, inArray } from "drizzle-orm";

export async function buildDiscoveryReport(caseId: string) {
  // 1) Case metadata
  const [caseRow] = await db
    .select()
    .from(cases)
    .where(eq(cases.id, caseId));

  if (!caseRow) throw new Error("Case not found");

  // 2) All communications (documents) for this case
  const docs = await db
    .select()
    .from(communications)
    .where(eq(communications.caseId, caseId));

  const docIds = docs.map((d) => d.id);
  const totalDocuments = docs.length;

  // 3) Codings: responsiveness + privilege
  const codings = docIds.length
    ? await db
        .select()
        .from(documentCodings)
        .where(inArray(documentCodings.documentId, docIds))
    : [];

  const codingByDocId = new Map<string, typeof codings[number][]>();
  for (const c of codings) {
    const arr = codingByDocId.get(c.documentId) ?? [];
    arr.push(c);
    codingByDocId.set(c.documentId, arr);
  }

  // 4) Tags
  const docTagRows = docIds.length
    ? await db
        .select({
          documentId: documentTags.entityId,
          tagId: documentTags.tagId,
          tagName: tags.name,
          tagCategory: tags.category,
          taggedBy: documentTags.taggedBy,
        })
        .from(documentTags)
        .innerJoin(tags, eq(tags.id, documentTags.tagId))
        .where(
          and(
            eq(documentTags.entityType, "communication"),
            inArray(documentTags.entityId, docIds)
          )
        )
    : [];

  // 5) Custodians for this case
  const custRows = await db
    .select()
    .from(custodians)
    .where(eq(custodians.caseId, caseId));

  const custodiansById = new Map(custRows.map((c) => [c.id, c]));

  // 6) Production sets for this case
  const prodSets = await db
    .select()
    .from(productionSets)
    .where(eq(productionSets.caseId, caseId));

  // 7) Privilege logs for this case
  const privLogs = await db
    .select()
    .from(privilegeLogs)
    .where(eq(privilegeLogs.caseId, caseId));

  // ---------- METADATA ----------
  const timestamps = docs
    .map((d) => d.timestamp)
    .filter(Boolean) as Date[];

  const earliestDate =
    timestamps.length && new Date(Math.min(...timestamps.map((d) => d.getTime())));
  const latestDate =
    timestamps.length && new Date(Math.max(...timestamps.map((d) => d.getTime())));

  const responsiveDocIds = new Set(
    codings
      .filter((c) => c.responsiveness === "responsive")
      .map((c) => c.documentId)
  );
  const privilegedDocIds = new Set(
    codings
      .filter((c) => c.isPrivileged === "true")
      .map((c) => c.documentId)
  );
  // also add comms with privilegeStatus != 'none'
  for (const d of docs) {
    if (d.privilegeStatus && d.privilegeStatus !== "none") {
      privilegedDocIds.add(d.id);
    }
  }

  const producedDocs = docs.filter(
    (d) => d.productionStatus === "produced"
  );

  const metadata = {
    caseId: caseRow.id,
    caseNumber: caseRow.caseNumber,
    caseName: caseRow.title,
    violationType: caseRow.violationType,
    totalDocuments,
    processedDocuments: totalDocuments,
    responsiveCount: responsiveDocIds.size,
    responsivePercent:
      totalDocuments > 0 ? (responsiveDocIds.size / totalDocuments) * 100 : 0,
    privilegedCount: privilegedDocIds.size,
    privilegedPercent:
      totalDocuments > 0 ? (privilegedDocIds.size / totalDocuments) * 100 : 0,
    totalProduced: producedDocs.length,
    dateRange:
      earliestDate && latestDate
        ? `${earliestDate.toISOString().slice(0, 10)} – ${latestDate
            .toISOString()
            .slice(0, 10)}`
        : "N/A",
    earliestDate: earliestDate?.toISOString() ?? null,
    latestDate: latestDate?.toISOString() ?? null,
  };

  // ---------- CUSTODIAN SUMMARY ----------
  const custodianMap: Record<
    string,
    {
      name: string;
      total: number;
      responsive: number;
      privileged: number;
      produced: number;
      department?: string | null;
    }
  > = {};

  for (const d of docs) {
    const key = d.custodianId || d.custodianName || "Unknown";
    if (!custodianMap[key]) {
      const cust =
        d.custodianId && custodiansById.get(d.custodianId as string);
      custodianMap[key] = {
        name: cust?.fullName || d.custodianName || "Unknown",
        total: 0,
        responsive: 0,
        privileged: 0,
        produced: 0,
        department: cust?.department ?? d.custodianDepartment,
      };
    }
    const c = custodianMap[key];
    c.total++;
    if (responsiveDocIds.has(d.id)) c.responsive++;
    if (privilegedDocIds.has(d.id)) c.privileged++;
    if (d.productionStatus === "produced") c.produced++;
  }

  const custodianSummary = Object.values(custodianMap);

  // ---------- TAG ANALYSIS ----------
  const tagMap: Record<
    string,
    { tag: string; category: string; count: number; appliedBy: Set<string> }
  > = {};
  for (const dt of docTagRows) {
    const key = dt.tagName;
    if (!tagMap[key]) {
      tagMap[key] = {
        tag: dt.tagName,
        category: dt.tagCategory,
        count: 0,
        appliedBy: new Set(),
      };
    }
    tagMap[key].count++;
    if (dt.taggedBy) tagMap[key].appliedBy.add(dt.taggedBy);
  }

  const tagAnalysis = Object.values(tagMap).map((t) => ({
    tag: t.tag,
    category: t.category,
    count: t.count,
    percent: totalDocuments ? (t.count / totalDocuments) * 100 : 0,
    appliedBy: [...t.appliedBy].join("; "),
  }));

  // ---------- PRIVILEGE SUMMARY ----------
  const privilegeSummary = {
    ac: 0,
    wp: 0,
    both: 0,
    risk: 0,
  };

  for (const c of codings) {
    if (c.isPrivileged === "true") {
      if (c.privilegeType === "attorney_client_privileged") privilegeSummary.ac++;
      else if (c.privilegeType === "work_product") privilegeSummary.wp++;
      else if (c.privilegeType === "both") privilegeSummary.both++;
    }
  }

  // risk: privileged docs with productionStatus = produced, or from privilegeLogs
  const privilegedProduced = docs.filter(
    (d) => privilegedDocIds.has(d.id) && d.productionStatus === "produced"
  ).length;
  privilegeSummary.risk = privilegedProduced + privLogs.length;

  // ---------- FILE TYPES ----------
  const ftMap: Record<
    string,
    { type: string; count: number; sizeBytes: number; examples: string[] }
  > = {};
  for (const d of docs) {
    const key = d.fileExtension || d.mimeType || d.communicationType || "unknown";
    if (!ftMap[key]) {
      ftMap[key] = { type: key, count: 0, sizeBytes: 0, examples: [] };
    }
    const f = ftMap[key];
    f.count++;
    f.sizeBytes += d.fileSize ?? 0;
    if (f.examples.length < 3) f.examples.push(d.subject || d.filePath || d.id);
  }
  const fileTypes = Object.values(ftMap).map((f) => ({
    type: f.type,
    count: f.count,
    sizeMb: f.sizeBytes / 1024 / 1024,
    examples: f.examples.join(", "),
  }));

  // ---------- PRODUCTION BATCHES ----------
  const productionBatches = prodSets.map((ps) => {
    const docsInSet = docs.filter(
      (d) => d.productionSetId === ps.id
    );
    const produced = docsInSet.filter(
      (d) => d.productionStatus === "produced"
    ).length;
    const withheld = docsInSet.filter(
      (d) => d.productionStatus === "withheld"
    ).length;
    const redacted = docsInSet.filter(
      (d) => d.isRedacted === "true"
    ).length;

    return {
      id: ps.id,
      number: ps.productionNumber,
      name: ps.productionName,
      date: ps.transmittedAt?.toISOString().slice(0, 10) ?? "",
      produced,
      withheld,
      redacted,
      format: ps.exportFormat,
      hash: JSON.stringify(ps.hashManifest ?? {}),
    };
  });

  // ---------- INVENTORY ----------
  const tagsByDocId = new Map<string, string[]>();
  const privTagsByDocId = new Map<string, string[]>();
  for (const dt of docTagRows) {
    const arr = tagsByDocId.get(dt.documentId) ?? [];
    arr.push(dt.tagName);
    tagsByDocId.set(dt.documentId, arr);
  }

  // you can also add privilege log info per doc if desired

  const inventory = docs.map((d) => ({
    id: d.id,
    caseId: d.caseId,
    custodian: d.custodianName,
    sender: d.sender,
    recipients: d.recipients ?? [],
    subject: d.subject,
    date: d.timestamp?.toISOString() ?? "",
    filePath: d.filePath,
    fileExtension: d.fileExtension,
    mimeType: d.mimeType,
    sizeKb: d.fileSize ? d.fileSize / 1024 : null,
    hash: d.documentHash,
    nearDuplicateHash: d.nearDuplicateHash,
    threadId: d.emailThreadId,
    familyId: d.documentFamilyId,
    productionStatus: d.productionStatus,
    batesNumber: d.batesNumber,
    batesRange: d.batesRange,
    tags: tagsByDocId.get(d.id) ?? [],
    privilegeStatus: d.privilegeStatus,
  }));

  return {
    metadata,
    custodianSummary,
    tagAnalysis,
    privilegeSummary,
    fileTypes,
    productionBatches,
    inventory,
  };
}
