# Sentinel Counsel LLP - Data Room Access Control & Permissions

## Current State

The Data Rooms feature currently allows:
- Creating data rooms linked to deals
- Organizing documents in a hierarchical folder structure
- Uploading documents to folders

## Required Enhancements

We need to add:
1. **External party access** - Invite third parties via email with secure links
2. **Granular folder/file permissions** - Control who sees what at folder and file level
3. **Action-based permissions** - Separate view, download, upload, and admin rights
4. **Permission groups** - Organize users into groups for easier management
5. **Access tracking & audit** - Log who accessed what and when

        </TooltipProvider>
        
        {/* Access Count Badge */}
        {permissionSummary.restrictedGroups > 0 && (
          <Badge variant="outline" className="text-xs h-5 px-1.5">
            {permissionSummary.accessibleGroups}/{permissionSummary.totalGroups}
          </Badge>
        )}
      </div>
      
      {/* Children */}
      {expanded && folder.children?.map((child) => (
        <FolderTreeItem 
          key={child.id}
          folder={child}
          level={level + 1}
          onSelect={onSelect}
          isSelected={selectedId === child.id}
        />
      ))}
      
      {/* Folder Permissions Dialog */}
      <FolderPermissionsDialog
        open={showPermissions}
        onClose={() => setShowPermissions(false)}

      />

function ActivityItem({ activity }: { activity: DataRoomActivity }) {

      case 'download': return <Download className="h-4 w-4 text-green-500" />;
